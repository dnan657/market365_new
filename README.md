# UK Classifieds — Полноценная платформа доски объявлений для Великобритании

UK Classifieds — это современная, высокопроизводительная платформа для размещения объявлений, созданная специально для британского рынка. Проект вдохновлен такими сервисами, как 999.md, Avito и OLX, и адаптирован под специфику Великобритании (валюта GBP £, британские регионы и почтовые индексы).

## 🚀 Основные функции

-   **Система объявлений**: Создание, редактирование и просмотр объявлений с поддержкой категорий (Авто, Недвижимость, Электроника и др.).
-   **Загрузка изображений**: Полноценная загрузка нескольких фотографий через API с локальным хранением.
-   **Интерактивные карты**: Отображение местоположения товара на карте (Leaflet) с поддержкой почтовых индексов Великобритании.
-   **Панель администратора**: Полный контроль над платформой — статистика выручки, модерация объявлений и управление пользователями.
-   **Умная модерация (AI)**: Автоматическая проверка контента на наличие спама, запрещенных ключевых слов и подозрительных цен (анти-скам).
-   **Монетизация**:
    *   **Продвижение (Promoted Ads)**: Платное поднятие объявлений в топ (симуляция оплаты).
    *   **Подписки**: Пакеты для профессиональных продавцов и бизнеса.
-   **Система поддержки**: Интерфейс для создания тикетов и связи с техподдержкой.
-   **Локализация**: Полная адаптация под UK — формат дат (en-GB), символ £, база городов Великобритании.

## 🛠 Технологический стек

-   **Frontend/Backend**: Next.js 15 (App Router)
-   **Язык**: TypeScript
-   **Стилизация**: Tailwind CSS
-   **База данных/ORM**: Prisma с SQLite (для легкой развертки)
-   **Иконки**: Lucide React
-   **Карты**: Leaflet / React Leaflet

## 📖 Принцип работы

1.  **Для пользователей**: После регистрации пользователи могут подавать объявления. Каждое объявление проходит через систему **AI-модерации**. Если система обнаруживает подозрительный контент, объявление помечается как `FLAGGED` для проверки администратором, иначе оно попадает в очередь `PENDING`.
2.  **Модерация**: Администратор в своей панели (`/admin`) одобряет или отклоняет новые публикации.
3.  **Поиск**: Реализован полнотекстовый поиск и фильтрация по регионам (Лондон, Манчестер и др.), категориям и диапазону цен.
4.  **Продвижение**: Пользователь может оплатить услугу "Featured", чтобы выделить свое объявление яркой рамкой и поднять его в начало списка.

## 📥 Установка и запуск

### Предварительные требования
Убедитесь, что у вас установлены **Node.js (v18+)** и **npm**.

---

### Инструкция для Linux / macOS

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/dnan657/market365_new.git
    cd market365_new
    ```

2.  **Установите зависимости**:
    ```bash
    npm install
    ```

3.  **Настройте базу данных**:
    ```bash
    npx prisma generate
    npx prisma db push
    ```

4.  **Запустите проект**:
    ```bash
    npm run dev
    ```
    Приложение будет доступно по адресу `http://localhost:3000`.

---

### Инструкция для Windows

1.  **Клонируйте репозиторий**:
    Используйте Git Bash или командную строку:
    ```cmd
    git clone https://github.com/dnan657/market365_new.git
    cd market365_new
    ```

2.  **Установите зависимости**:
    ```cmd
    npm install
    ```

3.  **Инициализируйте Prisma**:
    ```cmd
    npx prisma generate
    npx prisma db push
    ```

4.  **Запустите сервер разработки**:
    ```cmd
    npm run dev
    ```
    Откройте браузер и перейдите на `http://localhost:3000`.

---
