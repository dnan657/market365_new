# UK Classifieds — Полноценная платформа доски объявлений для Великобритании

UK Classifieds — это современная, высокопроизводительная платформа для размещения объявлений, созданная специально для британского рынка. Проект вдохновлен такими сервисами, как 999.md, Avito и OLX, и адаптирован под специфику Великобритании (валюта GBP £, британские регионы и почтовые индексы).

## 🚀 Основные функции

-   **Система объявлений**: Создание, редактирование и просмотр объявлений с поддержкой категорий (Авто, Недвижимость, Электроника и др.).
-   **Загрузка изображений**: Полноценная загрузка нескольких фотографий через API с локальным хранением и сохранением данных через Docker Volumes.
-   **Интерактивные карты**: Отображение местоположения товара на карте (Leaflet) с поддержкой почтовых индексов Великобритании.
-   **Панель администратора**: Полный контроль над платформой — статистика выручки, модерация объявлений и управление пользователями. Доступ защищен паролем.
-   **Умная модерация (AI)**: Автоматическая проверка контента на наличие спама, запрещенных ключевых слов и подозрительных цен (анти-скам) с использованием логики OpenAI Moderation API.
-   **Монетизация**:
    *   **Продвижение (Promoted Ads)**: Платное поднятие объявлений в топ (симуляция оплаты).
    *   **Подписки**: Пакеты для профессиональных продавцов и бизнеса.
-   **Система поддержки**: Полноценная тикет-система с сохранением в БД и управлением через админку.
-   **Локализация**: Полная адаптация под UK — формат дат (en-GB), символ £, база городов Великобритании.

## 🛠 Технологический стек

-   **Frontend/Backend**: Next.js 15 (App Router)
-   **Язык**: TypeScript
-   **Стилизация**: Tailwind CSS
-   **База данных/ORM**: Prisma с PostgreSQL
-   **Контейнеризация**: Docker & Docker Compose
-   **Иконки**: Lucide React
-   **Карты**: Leaflet / React Leaflet

## 📖 Принцип работы

1.  **Для пользователей**: После регистрации пользователи могут подавать объявления. Каждое объявление проходит через систему **AI-модерации**. Если система обнаруживает подозрительный контент, объявление помечается как `FLAGGED` для проверки администратором, иначе оно попадает в очередь `PENDING`.
2.  **Модерация**: Администратор в своей панели (`/admin`) одобряет или отклоняет новые публикации.
3.  **Поиск**: Реализован полнотекстовый поиск и фильтрация по регионам (Лондон, Манчестер и др.), категориям и диапазону цен.
4.  **Продвижение**: Пользователь может оплатить услугу "Featured", чтобы выделить свое объявление яркой рамкой и поднять его в начало списка.

## 🐳 Развертывание через Docker (Рекомендуется)

Самый быстрый способ запустить платформу — использовать Docker Compose. Это автоматически поднимет базу данных PostgreSQL и веб-приложение.

1.  **Убедитесь, что у вас установлен Docker и Docker Compose**.
2.  **Запустите проект одной командой**:
    ```bash
    docker compose up --build
    ```
3.  **Инициализируйте базу данных** (выполните в новом окне терминала после запуска контейнеров):
    ```bash
    docker compose exec app npx prisma db push
    docker compose exec app npx ts-node prisma/seed.ts
    ```
4.  **Панель администратора**:
    -   Адрес: `http://localhost:3000/admin`
    -   Пароль по умолчанию: `uk-admin-2026` (настраивается в `docker-compose.yml`)

## 📥 Локальная установка (без Docker)

### Предварительные требования
Убедитесь, что у вас установлен **Node.js (v18+)**, **npm** и запущенный сервер **PostgreSQL**.

1.  **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/dnan657/market365_new.git
    cd market365_new
    ```

2.  **Настройте переменные окружения**:
    Создайте файл `.env` и укажите в нем `DATABASE_URL` для вашего PostgreSQL.
    ```env
    DATABASE_URL="postgresql://user:password@localhost:5432/ukclassifieds?schema=public"
    ```

3.  **Установите зависимости и запустите проект**:
    ```bash
    npm install
    npx prisma generate
    npx prisma db push
    npx ts-node prisma/seed.ts
    npm run dev
    ```
